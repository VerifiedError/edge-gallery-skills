# Sunrise & Sunset

This skill returns today's sunrise, sunset, solar noon, and total daylight duration at the user's location using the free sunrise-sunset.org API. No API key required.

## Examples

- "When does the sun rise today?"
- "What time is sunset?"
- "How many hours of daylight do I have today?"
- "When is solar noon?"

## Instructions

Call the `run_js` tool with the following exact parameters:

- data: A JSON string. Pass `{}`.
